# 🎓 Student Performance Prediction

## 📖 Overview
A Machine Learning web application that predicts a student's exam score based on academic, personal, and environmental factors using a trained regression model.

## ✨ Features
- Predicts exam scores in real time.
- Interactive Streamlit interface.
- User-friendly input controls.
- Fast and accurate predictions.

## 🛠️ Tech Stack
- Python
- Pandas
- NumPy
- Scikit-learn
- Streamlit
- Joblib

## 📊 Input Features
- Hours Studied
- Attendance
- Sleep Hours
- Previous Scores
- Tutoring Sessions
- Physical Activity
- Parental Involvement
- Access to Resources
- Motivation Level
- Internet Access
- Family Income
- Teacher Quality
- School Type
- Peer Influence
- Learning Disabilities
- Parental Education Level
- Distance from Home
- Gender
